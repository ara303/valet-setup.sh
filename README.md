# valet-setup.sh
A setup script for what Larvel Valet doesn't give you for PHP development.

## Needs:
- A Mac (probably only Apple Silicon)
- [Homebrew](https://brew.sh/)

## Does:

- What Valet doesn't, specifically: 
  - Mariadb
  - Mailhog
  - Xdebug

## Gives:

A few handy `.env` variables and where to go to view Mailhog's emails as well as Xdebug's adapter endpoint.

## Status:

WIP. I made this because Valet Plus is abandonware, but I still like Valet the most for setting up PHP environments. I regularly need to easily spin up
